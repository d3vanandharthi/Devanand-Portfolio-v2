
# Devanand Harthi - Futuristic Portfolio

A high-performance, interactive portfolio website built with React, TypeScript, Tailwind CSS, and Gemini AI.

## Features
- **Futuristic UI/UX**: Inspired by lusion.co, featuring WebGL-style particle backgrounds and custom cursor interactions.
- **AI Chat Assistant**: Integrated with Google Gemini 2.5 Flash for real-time interaction about professional experience.
- **Data Visualization**: D3.js force-directed graph for skills.
- **AI Gesture Control**: "Minority Report" style webcam interface using MediaPipe.
- **Performance**: Optimized React code with Framer Motion and lightweight animations.

## AI Gesture Controls
Experience the future of browsing without a mouse. Click "ENABLE GESTURE CONTROL" in the bottom-left corner to activate.

| Gesture | Action | Description |
|---------|--------|-------------|
| **Point** | **Move Cursor** | Raise your index finger. The cursor follows your fingertip. |
| **Pinch** | **Click** | Pinch your index finger and thumb together to click items. |
| **Fist** | **Scroll** | Make a closed fist and move your hand up/down to drag the page. |

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS
- Google GenAI SDK
- Google MediaPipe (Vision)
- D3.js
- Lucide React

## Getting Started

1. Clone the repository.
2. Create a `.env` file with your `API_KEY` for Google Gemini.
3. Open `index.html` in a modern browser (or serve via a local server).
